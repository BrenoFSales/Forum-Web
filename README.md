## Requerimentos

## Como rodar
### Utilizando o terminal:
```sh
git clone https://github.com/BrenoFSales/Forum-Web/
cd Forum-Web
python -m venv .venv # imponrtante que seja .venv
pip install -r requirements.txt
pip install flask
flask run
```
